🌐 Portfolio Websites Collection

A collection of modern, responsive website templates built with HTML, CSS, and JavaScript, designed to help businesses quickly establish a professional online presence.

These templates are created to be ready-to-use demos for client pitching and freelance work.

🚀 Purpose

This repository is not just for code — it’s a client showcase toolkit.

Each project here is designed to:

Demonstrate real business website ideas
Be used as live demos when pitching clients
Help close deals faster by showing working prototypes
Save time by reusing and customizing templates
🧩 What’s Inside

You will find different types of business-ready templates such as:

🏢 Business / Corporate landing pages
🛒 Product or service landing pages
🍽️ Restaurant websites
💇 Salon / local service websites
📊 Portfolio / personal branding pages

Each template includes:

Responsive design (mobile-friendly 📱)
Clean UI/UX
Simple and editable code structure
Fast loading static pages
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
Responsive Design (Flexbox / Grid)
📂 How to Use
Clone the repository:
git clone https://github.com/Amar-elhachemi/portfolio-websites.git
